# 建筑图纸设计依据/施工图审查依据 - 广东/广州层面

检索日期：2026-05-21

用途：适用于广州项目的建筑图纸设计依据、施工图审查依据和地方政策检索。

## 广东/广州层面建议补充

1. 《广东省建设工程质量管理条例》（2023修正）
   - 来源：广东省政府
   - URL：https://www.gd.gov.cn/zwgk/wjk/zcfgk/content/post_2532382.html
2. 《广东省建设工程勘察设计管理条例》
   - 来源：广东省政府
   - URL：https://www.gd.gov.cn/zwgk/wjk/zcfgk/content/post_2521224.html
3. 广东省施工图数字化审查系统相关要求
   - 来源：广东省住建厅
   - URL：https://zfcxjst.gd.gov.cn/gkmlpt/content/2/2640/post_2640794.html?jump=false
4. 《广州市城乡规划技术规定》
   - 涉及内容：用地性质、容积率、建筑密度、绿地率、建筑间距、建筑退让、停车配建等规划控制内容。
   - 来源：广州市政府
   - URL：https://www.gz.gov.cn/zwgk/fggw/gz/content/post_7953362.html
5. 《广州市城乡规划程序规定》
   - 涉及内容：建设工程设计方案、总平面图、电子图纸等规划许可材料。
   - 来源：广州市规划和自然资源局
   - URL：https://ghzyj.gz.gov.cn/sjb/zw/zcfg/content/post_9466528.html
6. 《广州市房屋建筑和市政基础设施工程质量管理办法》
   - 施行日期：2025-01-01。
   - 重点：要求施工图联合审查、设计文件符合设计深度和强制性标准。
   - 来源：广州市政府
   - URL：https://www.gz.gov.cn/zwgk/fggw/gz/content/post_10045043.html
7. 广州施工图审查最新要求
   - 重点：可整体申报或分阶段申报，分阶段审查后的图纸作为监管依据。
   - 来源：广州市住建局
   - URL：https://zfcj.gz.gov.cn/zjdt/ztzl/gzsgcjsxmspzdgggzzl/bszy/zcwj/content/post_10754111.html
8. 广州已清理部分施工图审查内容
   - 重点：原则上不得新增无合法依据的审查事项。
   - 来源：广州市住建局
   - URL：https://zfcj.gz.gov.cn/zjyw/kjsj/jskj/content/mpost_10059274.html

## 广州项目图纸依据建议

广州项目在国家层面概括句之外，建议单独列入：

- 《广州市城乡规划技术规定》
- 《广州市房屋建筑和市政基础设施工程质量管理办法》
