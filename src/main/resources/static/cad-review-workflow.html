1. 创建运行

上传 DWG/DXF，生成 runId，主表写入 UPLOADED。

  • ReviewRunController
  • ReviewRunService.create
  • MultipartFileSnapshot

2. 解析与清洗

调用外部 CAD 解析器，保存 raw IR，再构造 review_context。

  • CadParserService
  • CadIrCleaner
  • raw_ir_json / clean_context_json

3. 规则与任务

规则摘要进入 Dispatcher，输出 ReviewTask 并按优先级限流。

  • RegulationPlannerAgent
  • DispatcherAgent
  • maxReviewTasks

4. 并行审核

每个任务单独构造证据包，Reviewer 并行产出 Finding。

  • TaskContextBuilder
  • PreCleanerAgent
  • ReviewerAgent

5. 汇总报告

汇总所有 Finding，检测冲突，计算覆盖率并生成 ReviewReport。

  • SummarizerAgent
  • ConflictGroup
  • ReviewCoverage

6. 上下文治理

每次模型调用前压缩上下文，记录超限策略和丢弃数量。

  • ContextBudgetService
  • ContextEnvelope
  • SHRINK / TOP_K / SPLIT

7. 结果落库

任务状态和最终报告写入数据库，通过 runId 查询。

  • cad_review_runs
  • cad_review_run_tasks
  • report_json

8. 降级兜底

异常、超时、证据不足进入 FAILED、PARTIAL 或 PENDING_REVIEW。

  • 失败任务保留原因
  • 跳过任务保留 ruleId
  • 不强行 PASS/FAIL
